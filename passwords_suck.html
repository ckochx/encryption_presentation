    • What can we do today? (Personal)

    Short answer: There are no widely available alternatives out there.

    Intermediate steps:

    Use a well-vetted password manager (like LastPass) and generate strong passwords for all your logins.

    Generate the longest password allowed by the service

    Use 2-factor authentication

    What else can we do today? (NCSA)

    Use ssh with keys for server login

    Keep the password around to elevate acess as needed

    Don't allow password-based login to servers

    How to SSH It's easy 
    ssh username@host.extension

    When you login you will be prompted to enter the password associated with that username.


    But we can ssh without passwords

    Awesome, How do I do this? It's easy 
    ssh-keygen -t ecdsa -b 521 -C "you_email@ncsasports.org"

    OR

    ssh-keygen -t ed25519 -b 521 -C "you_email@ncsasports.org"

    OR

    ssh-keygen -t rsa -b 4096 -C "you_email@ncsasports.org"

    *Always use the largest bit value available.

    **Different algorithms have differnet max bit values
